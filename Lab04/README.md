# simple-blog

Simple web application for blog posting using **React** and **ASP.NET WebAPI**.

**simple-blog-api** - all the business logic is here

**simple-blog-app** - Front-end for project

## Prequisities

- Visual Studio with tools for ASP.Net Development
- Node.js 14.x.x LTS
- yarn

## How to start project

1. Open *simple-blog.sln* in Visual Studio and start IIS Express by clicking this button
![Start IIS Express button](misc/start-IIS.png)

2. Open terminal inside *simple-blog-app* folder and run `yarn start`
